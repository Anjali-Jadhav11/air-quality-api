# Air Quality Index (AQI) Search Engine

## Description
Spring Boot + HTML/JS project to search AQI by city.

## Features
- Fetches real-time AQICN data
- Shows AQI, PM2.5, PM10, and dominant pollutant
- Color-coded AQI levels
- PM2.5/PM10 bar chart
- Handles invalid city or token gracefully
